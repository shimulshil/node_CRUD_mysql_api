Build a REST API in Node js & Express:
1.	Create a folder like- NODE_EXPRESS_API
2.	npm init --y     It will create package.json file
3.	npm install express --save  
4.	npm install body-parser --save  
5.	npm install mysql --save  
6.	create index.js file
7.	npm install --save-dev nodemon
8.	add this line in package.json file line -8: “start”: “nodemon index”
9.	in terminal : npm start
10.	check the node version in terminal: node –v
11.	add a new line inside the package.json: “type”: “Module” , in line number -06
